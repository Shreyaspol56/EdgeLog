PC Setup Instructions
01 Download the trade-journal.html file to your PC
02 Double-click it — opens in Chrome, Edge, or Firefox (no install needed)
03 To make it a desktop app in Chrome: Click ⋮ menu → Save and share → Install page as app
04 It appears in your Start Menu and taskbar like a real app
📱 Android setup: Email the HTML file to yourself → open in Chrome → tap ⋮ → Add to Home Screen
🔄 Keeping devices in sync: Use the Export/Import backup above, or save the HTML file to Google Drive and open it there on both devices
